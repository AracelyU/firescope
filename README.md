# Comandos a ejecutar para descargar los datos de data_download.py

```bash
Descargar los datos de Topología (SRTM)
python src/data_download.py --sources srtm

Descargar los datos de Vegetación (Sentinel-2)
python src/data_download.py --sources sentinel2

Descargar los datos de Viento (ERA5)
python src/data_download.py --sources era5

Descargar los datos de Análisis temporal de incendios (PANGAEA)
python src/data_download.py --sources pangaea

Descargar los datos de CONAF
python src/data_download.py --sources conaf

Descargar todos los datos 
python src/data_download.py --sources all
```
